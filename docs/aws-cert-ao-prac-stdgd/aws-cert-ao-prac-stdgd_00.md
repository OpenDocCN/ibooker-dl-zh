# 前言

人工智能（AI）是我们这一代人中最具变革性的技术之一。超过 25 年来，亚马逊一直在投资人工智能和机器学习（ML）的发展，因为我们相信这些技术具有改变我们客户和社区生活的力量。

随着生成式人工智能的兴起，我们看到越来越多的组织采用人工智能服务来推动创新和释放非凡的能力。但尽管人工智能使创新成为可能，是人们使它成为现实。正是这个人的因素，使得组织积极寻求人工智能人才。[我们的研究](https://oreil.ly/0g7Ww)显示，92%的组织有计划雇佣具有人工智能技能的人来帮助他们推动人工智能创新。虽然这是一个好消息，但目前全球人工智能人才短缺。[四分之三的雇主](https://oreil.ly/2rPhg)认为雇佣具有人工智能技能的人才是一个优先事项，但找不到他们需要的人工智能人才。

因此，在 2023 年，我们承诺到 2025 年通过亚马逊的 AI Ready 计划，为全球两百万名人士提供免费的人工智能技能培训。通过这一承诺，学习者将他们的知识应用于工作，帮助他们提高生产力和创造力，向潜在雇主展示他们的技能，帮助他们成为更有竞争力的候选人，或者追求进一步培训，以在人工智能领域建立职业生涯。

我们提前一年实现了 AI Ready 的目标，但这只是开始。我们将继续帮助更多的人在人工智能领域提升技能，因为人工智能技能已经成为现代劳动力不可或缺的一部分。投资于人工智能技能培训的组织和个人将更好地适应这一领域的快速进步，并在前所未有的技术变革时代蓬勃发展。

我很高兴你通过追求 AWS 认证人工智能实践者来开始人工智能技能提升之旅。汤姆的书籍是 AWS 提供的考试准备之外的绝佳资源，帮助你为考试做准备。这是一本全面的指南，涵盖了 AWS、AWS 人工智能和机器学习服务及工具的基础知识；负责任的人工智能；帮助通过考试的策略；以及更多内容。

在我的整个职业生涯中，我在每个阶段都学会了新的技能，为与我合作的组织创造价值。我坚信，好奇心和对终身学习的承诺也将是推动你事业发展的关键。无论这是你获得的第一张证书还是工具箱中的许多证书之一，你都在展示这些价值观，这将为你准备一个由人工智能驱动的未来。你对人工智能技能提升的投资不仅将赋予你为自己建立一个持久而有意义的职业生涯的能力，还将解锁对未来的几代人有益的激动人心的创新。
