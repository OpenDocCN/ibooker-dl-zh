# 第一部分. 为数据科学构建 API

本书的第一部分通过示例项目逐步指导你使用 Python 和 FastAPI 构建 API 并将其部署到云端：

+   在第一章，你将通过理解用户需求并选择用例来满足这些需求，开始你的项目组合。

+   在第二章，你将选择一个 API 架构，并开始创建一个 API 以满足数据科学家的需求。

+   在第三章，你将使用 SQLite 创建数据库，创建使用 SQLAlchemy 读取数据库的 Python 代码，并使用 Pytest 进行单元测试。

+   在第四章，你将创建 FastAPI 代码来使用这些数据，并将其发布为 REST API。

+   在第五章，你将使用 FastAPI 内置的功能来记录你的 API。

+   在第六章，你将使用 Render 和亚马逊网络服务（AWS）将你的 API 部署到云端。

+   在第七章，你将创建一个软件开发工具包（SDK），以便 Python 开发者更容易使用你的 API。
