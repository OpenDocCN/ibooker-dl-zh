# 第三部分：交付数据故事

数据、信息、知识、智慧（DIKW）金字塔以及生成式 AI 的使用已帮助您构建了一个针对受众的数据故事。首先，您通过使用不同的技术，如联系、巧合、好奇心和矛盾，提取有趣的见解，将数据转化为信息。您使用 Altair 和 GitHub Copilot 构建了一个基本图表来表示提取的见解。接下来，您通过添加图表的上下文（可以是文本或视觉的）将信息转化为知识。您使用 ChatGPT 生成文本上下文，使用 DALL-E 生成视觉上下文。最后，您通过在故事中添加下一步行动，将知识转化为智慧。您了解到存在不同类型的下一步行动，包括意识提升和活动、协作努力、倡导和政策、培训和开发、战略行动以及参与和沟通。每一项都取决于您想要实现的具体目标。

您的数据故事现在已准备就绪，但您的工作尚未完成。您有责任以道德的方式讲述您的故事，这意味着您应该考虑文化、传统以及所有受众所信任的框架。接下来，您就可以发布您的故事了。

在第十章中，您将考虑使用生成式 AI 时常见的议题，包括幻觉、偏见和版权。您还将学习联合国教科文组织关于监控生成式 AI 输出的指南，以确保其不会歧视任何人。

在第十一章中，您将学习使用 Streamlit、Tableau、Power BI 和 Comet 等工具发布数据故事的热门策略。您将使用 Streamlit 作为一个独立的框架来构建网络应用程序。您还将学习如何将您的数据故事整合到两个流行的数据故事讲述工具：Tableau 和 Power BI。最后，您将使用 Comet，这是一个机器学习实验平台，同时也提供报告构建系统。最后，您将了解如何将单一图表的数据故事拆分为多个幻灯片，以便进行现场演示。
