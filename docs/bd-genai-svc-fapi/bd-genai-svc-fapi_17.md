# 后记

感谢所有加入我在这段通过 FastAPI 构建生成式人工智能服务世界的迷人旅程的人。我赞赏你们对理解构建内容丰富且适用于生产的生成式人工智能应用端到端过程的承诺。

在这本书中，我们探讨了使用 FastAPI 开发人工智能服务的发展，涵盖了从项目设置到跨各种模态集成和提供生成模型的所有内容。我们讨论了什么是生成式人工智能，以及为什么你们将在未来看到更多由生成式人工智能模型驱动的应用。接下来，你们学习了更多关于 FastAPI Web 框架以及如何创建类型安全的 AI 服务。作为其中的一部分，我们为我们的 AI 服务实现了并发、实时通信和数据库集成。最后，我们专注于保护、优化、测试和部署我们的 AI 服务，包括实现身份验证、授权和多种安全措施。

在掌握了更多关于这些技术的知识和经验之后，你现在可以构建自己的创新和稳健的生成式人工智能应用，在人工智能快速发展的领域中保持领先。

随着我们结束这本书的编写，我想强调的是，我们在构建生成式人工智能应用的学习之旅才刚刚开始。前方有许多令人兴奋的发展，我鼓励你们继续关注该领域的最新进展。为此，请关注本书的[仓库](https://github.com/Ali-Parandeh/building-generative-ai-services)和[网站](https://buildinggenai.com)，因为我将继续添加资源。

我希望这本书能让你对与 Web 服务集成的生成式人工智能模型及其在生产中改变行业的潜力有更深入的理解。以这本书为指南，我相信你将准备好在生成式人工智能服务的动态领域中导航，并为这个快速发展的领域做出重大贡献。
