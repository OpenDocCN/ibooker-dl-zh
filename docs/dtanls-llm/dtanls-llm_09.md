# 第三部分 高级主题

这一部分书籍将帮助你编写更有效、成本效益更高的语言模型应用程序。

第八章将我们的范围从 OpenAI 的语言模型扩展到其他提供商。在将语言模型应用于大数据集之前，比较不同提供商提供的模型至关重要。这样，你可以为特定场景获得成本和质量的最佳权衡。本章讨论了一些最受欢迎的提供商、他们的模型以及他们提供的库。

第九章在一个示例场景中展示了成本优化的技术。它涵盖了诸如提示工程、模型配置参数的最优调整以及微调等主题，微调是一种将语言模型专门用于一项特定任务的过程。正如本章所示，使用这些方法可以在质量和成本方面带来显著的改进。

第十章讨论了两个在语言模型上开发复杂应用的非常流行的软件框架：LangChain 和 LlamaIndex。这两个框架都可以用于数据分析。特别是，本章展示了如何使用这些框架来构建智能体，这种方法使语言模型能够使用各种计算工具解决复杂的数据分析任务。
