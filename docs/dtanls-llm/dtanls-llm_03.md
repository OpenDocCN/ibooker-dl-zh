# 第二部分：使用语言模型进行数据分析

对于小数据量的语言模型使用，一个网络界面就足够了。但如何将其扩展到大数据集呢？

在第三章中，我们探讨了 OpenAI 的 Python 库，这是一个强大的库，它使您能够直接从 Python 调用语言模型。我们讨论了最重要的库函数，包括可以用于微调语言模型行为的各种参数，以适应您的特定用例。

第四章讨论了几个使用 Python 库通过 OpenAI 的语言模型进行文本文档集合分析的小型项目。例如，我们的项目涵盖了将文本文档分类到预定义的几个类别之一，从文本中提取特定信息片段，以及将相似文本文档聚类在一起。

第五章展示了如何使用语言模型分析结构化数据：数据表和图数据。在这里，语言模型被用作接口，将关于数据的问题（以自然语言形式）转换为可以由专用分析工具处理的正式查询。例如，本章演示了如何使用语言模型构建将问题转换为 SQL 查询的界面，这些查询可以使用关系数据库管理系统进行处理。

第六章展示了语言模型如何分析图片甚至视频。示例项目包括使用语言模型以自然语言回答关于图片的任意问题，以及找到显示同一人的图片对。本章还展示了如何使用语言模型根据视频内容自动为视频添加字幕。

第七章专注于音频数据，涵盖了语音转录以及语音生成。例如，我们看到了如何使用 OpenAI 的模型构建一个语音查询界面，回答关于表格数据的语音问题。我们还构建了一个翻译工具，根据用户指定将一种语言的语音输入翻译成另一种语言的语音输出。
