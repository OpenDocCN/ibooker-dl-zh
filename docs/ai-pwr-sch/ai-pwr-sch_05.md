# 第二部分 学习特定领域意图

在第一部分，你学习了基于关键词（使用 TF-IDF 和 BM25）和数值向量（使用余弦和点积）的匹配和排名机制。你还了解到了众包相关性排名的概述。然而，在执行这些排名技术之前，能够正确解读用户的查询并运行一个理解用户意图的适当搜索是非常重要的。如果运行了错误的查询并且匹配了错误的文档，那么在这些不良结果上进行的任何排名逻辑都无法克服误解的查询。

正确解读用户的查询是第二部分的重点。这一步骤常常被忽视，因为它非常具有领域特定性，但对于满足用户的信息需求至关重要。在第五章，你将学习如何遍历语义知识图谱，这是在第二章中引入的包含在你搜索引擎内部的超结构化数据的巨大图。这些图构成了搜索引擎的基础，并能够推断出数据中特定领域术语的细微含义。在第六章，你将学习如何执行查询意图分类，使用这些图来消除词语和短语的歧义，以及如何利用你的内容和用户行为信号来学习特定领域的术语、相关术语、拼写错误和术语的变体形式。在第七章，我们将把所有你学到的知识结合起来，构建一个查询管道以执行语义搜索。这个管道将解析用户查询的特定领域意图，并将它们重写以向搜索引擎表示用户意图的更准确的语义解释，从而产生更加相关的搜索结果。
