# 附录 A 本书 GitHub 仓库

本书的不同章节对不同的代码片段解释得很好，但将各种代码样本一起运行具有挑战性。与其将所有代码拼凑在一起然后尝试运行，你可以在 GitHub 仓库中找到它们，本附录概述了详情。

如我们在各章节中看到的，本书的 GitHub 仓库有一个易于记忆的短网址：[`bit.ly/GenAIBook`](https://bit.ly/GenAIBook)。该网址重定向到仓库，可在[`github.com/bahree/GenAIBook`](https://github.com/bahree/GenAIBook)找到。

这个 GitHub 仓库提供了一些可能感兴趣的东西：

+   GitHub 仓库中的章节整洁地组织了来自书中不同示例的代码。文件名与章节中的列表编号相对应，这使得定位特定示例变得轻而易举。

+   各章节还讨论并引用了与各种人工智能技术和方法相关的学术论文。汇编按章节排列，便于访问相关的研究论文。

+   提供了详细的、最新的说明，用于在您的机器上设置依赖项以运行所有内容。鉴于该领域的快速变化，纸张并不是解释它们的理想媒介，因为它们很快就会过时。

+   有一个简单的本地可运行的网络应用程序，可以将许多这些概念结合起来，几乎可以成为你的个人 ChatGPT。

+   GitHub 仓库提供了讨论并指出问题，这促进了反馈并改善了在整个开发过程中的书籍和材料，尤其是在人工智能快速发展的背景下。
