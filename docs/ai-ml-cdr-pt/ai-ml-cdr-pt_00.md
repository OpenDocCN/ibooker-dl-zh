# 前言

亲爱的读者，

人工智能正准备改变每一个行业，但几乎每个人工智能应用都需要针对其特定的用途进行定制。用于阅读病历的系统与用于在工厂中寻找缺陷的系统不同，这与产品推荐引擎也不同。为了使人工智能充分发挥其潜力，工程师需要能够帮助他们适应数百万个具体问题所提供的惊人能力的工具。

当我领导谷歌大脑团队时，我们开始构建一个名为 DistBelief 的 C++ 深度学习框架。我们对于利用数千个 CPU 来训练神经网络（例如，使用 16,000 个 CPU 在未标记的 YouTube 视频上训练猫检测器）的潜力感到兴奋。自那时以来，深度学习已经取得了多么大的进步！曾经的前沿技术现在可以在你的笔记本电脑上完成！在这本书中，你将学习如何使用 PyTorch 来实现这一点。用于创建机器学习模型以实现人工智能的框架也已经取得了长足的进步。

PyTorch 取得了显著的进步。它旨在易于学习，同时足够强大，可供研究人员使用。其丰富的功能可以帮助人们构建各种人工智能工具，从简单的模型到从他人那里迁移学习，再到微调最现代的生成人工智能模型。

今天，数百万的程序员因为像 PyTorch 这样的框架而成为了人工智能开发者。

劳伦斯·莫罗尼，一直是帮助开发者成功使用 TensorFlow 和 PyTorch 进行人工智能的主要力量。我有幸与他一起在 deeplearning.ai 和 Coursera 教学几个专业课程，包括即将推出的 PyTorch 课程。
