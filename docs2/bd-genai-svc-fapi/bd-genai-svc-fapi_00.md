# 前言

我记得阿里，ADSP 的工程负责人，自信地走进办公室，宣布他想写一本关于创建生成式人工智能服务的书。鉴于撰写技术书籍是一项庞大的工作，我给他递了一杯浓咖啡，并讲述了我那些由咖啡因和纯粹意志驱动的夜间写作经历，这些经历是为了遵守截止日期。我甚至可能在他耳边低声提醒了凌晨 3 点重写整个章节的诱惑，但阿里坚定不移。他的眼中闪烁着光芒，那是决心和清晰愿景的混合。他知道他想创造一些特别的东西，一些能够揭示生成式人工智能复杂性的东西，并允许其他人构建它。

读完《使用 FastAPI 构建生成式人工智能应用》后，我可以肯定地说，它做了很多。阿里为那些想要超越人工智能理论讨论，进入实际应用世界的人创造了一本不可或缺的指南。在某种程度上，他让整个过程看起来出奇地简单。

作为一家人工智能咨询公司的联合创始人，我亲眼见证了工程师需求的日益增长，他们不仅需要能够理解人工智能的工作原理，还需要能够使用人工智能构建生产级别的解决方案。我们正处于一个深刻的转型时期，人工智能正在迅速改变我们的生活方式和工作方式。仅仅成为人工智能驱动的产品的被动消费者已经不再足够。未来属于那些知道如何利用生成模型的力量来创造、创新和解决实际问题的人。这本书是这一旅程的完美起点。

阿里的技术专长在每一页都显而易见。他能够轻松地将复杂的概念与清晰、简洁的解释和实际案例相结合。代码片段不仅仅是玩具示例，而是构建实际应用的构建块。他引导你穿越 FastAPI 的复杂性、身份验证、授权和数据库集成，就像一位经验丰富的工程师在现实世界中与这些挑战作斗争一样安全。

《使用 FastAPI 构建生成式人工智能应用》是所有试图在快速发展的人工智能领域找到方向的工程师的宝贵资源。这是阿里技术领导力的证明，以及他使复杂性易于理解的非凡能力。这本书不仅关乎创建人工智能服务，还关乎让新一代工程师塑造未来的可能性。而且，得益于这样的作品，那个未来看起来无比光明。

大卫·福斯特

ADSP 合作伙伴
