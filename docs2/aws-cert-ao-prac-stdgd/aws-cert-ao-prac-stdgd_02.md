# 第一章：为什么要认证？AWS 人工智能实践者考试简介

AWS 认证人工智能实践者（考试代码为 AIF-C01）认证专注于人工智能（AI）、机器学习（ML）和生成式人工智能的入门级。考试涵盖了众多 AWS 工具。

根据 AWS 培训和认证产品与服务总监 Jenni Troutman 的说法：

> AWS 认证人工智能实践者考试是一项基础人工智能认证，旨在让来自不同背景和经验的人展示他们对人工智能和生成式人工智能概念的理解，他们识别受益于人工智能的机会的能力，以及他们使用人工智能工具的负责知识。

如果您已经了解这些主题，这无疑会帮助您通过考试。但这不是必需的。本书将提供您需要了解的内容，分解考试中涵盖的概念。每个章节的末尾也有样题，以及一个练习考试。

在本章中，我们将探讨以下内容：

+   参加考试的原因

+   目标受众

+   考试涵盖的主题

+   其他可以考虑的认证

# 为什么你应该参加这个考试？

获得 AWS 认证人工智能实践者认证可以提升您的职业生涯。这在快速发展的 AI 行业中尤为重要。

让我们更深入地了解一下。

## 认可

该认证表明，您理解人工智能的基础知识，以及亚马逊的庞大云平台 AWS。您还可以在 LinkedIn 等网络中突出显示该认证，为您的个人资料添加数字徽章。

即使您不打算追求人工智能角色，该认证仍然很有价值。毕竟，在当今许多工作中——无论是销售、营销、法律等——通常都需要理解人工智能。

## 人工智能的增长

[Gartner 预测](https://oreil.ly/ged-3)，到 2027 年，人工智能软件的支出将达到 2979 亿美元，年增长率为 19.1%。企业一直在大力投资人工智能，以提高效率和生产力。但这项强大的技术也可以通过解锁企业数据中的洞察力来推动增长。

当然，这意味着企业将需要熟悉人工智能的员工。根据 AWS 委托的一项[研究](https://oreil.ly/LnhJq)，超过 90%的雇主预计在未来五年内采用与人工智能相关的解决方案。一些为 AWS 认证人工智能实践者（AWS 认证机器学习工程师（副高级）认证）培训员工的公司包括 Capgemini、Cognizant、Compass、Deloitte、DXC Technology、EPAM、Kyndryl、N-iX、SoftwareOne、Tata Consultancy Services 和 Wipro。

## AWS 继续占据主导地位

AWS 是全球云基础设施市场的领导者。截至 2024 年第一季度，其[市场份额达到 31%](https://oreil.ly/Q_wGF)。微软的市场份额为 25%，谷歌云为 10%。

2024 年第三季度，[AWS 收入达到 275 亿美元](https://oreil.ly/QbJVa)。这比去年同期增长了 19%。增长的核心是 AWS 的 AI 部门。对于 2024 年，其收入以[三位数速度](https://oreil.ly/cc2Xf)增长。

## 利用 AI 技能提升薪资、生产力和职业生涯

“拥有 AI 技能可以提高生产力，”特劳特曼说。“事实上，88%的员工预计到 2028 年将在日常工作中使用 AI 来自动化重复性任务并激发创造力。在我们的研究中，我们也发现，拥有 AWS 认证的员工与他们的技术同事和客户的互动得到了改善。”

鉴于这些好处，拥有 AI 技能的人的薪酬有了显著增长。考虑一下[2023 年 11 月的 AWS 研究](https://oreil.ly/lzppR)。它显示，对于销售和市场营销职位的人来说，他们的薪资通常比其他人高出 43%，对于财务职位的人来说高出 42%，对于业务运营职位的人来说高出 41%，而对于 IT 专家来说高出 47%。

那么，那些担任 AI 角色的人又如何呢？以下是根据[Coursera](https://oreil.ly/eRa7e)的数据，对美国平均薪酬水平的分析：

数据分析师

此人的起薪在 60,000 美元到 90,000 美元之间。这取决于个人的背景和地点。例如，旧金山的数据分析师的平均年薪为 95,000 美元。

AI 开发者

AI 开发者或机器学习工程师负责构建、部署和监控 AI 项目和模型。他们了解 Python 等语言以及 TensorFlow、PyTorch 或 scikit-learn 等框架。AI 开发者的年薪在 85,000 美元到 120,000 美元之间。

项目经理

项目经理的年薪在 90,000 美元到 130,000 美元之间。他们的薪资取决于他们的技能组合以及他们管理的项目的复杂性。

AI 解决方案销售专家

薪资通常在每年 70,000 美元到 110,000 美元之间，但由于佣金，薪酬可能远高于此。

## 提高经济增长和业务运营

[麦肯锡报告](https://oreil.ly/scIKI)估计，由于预期生产力和效率的提高，生成式 AI 系统将为全球年度 GDP 增加 2.6 万亿美元到 4.4 万亿美元。预计将受到最大影响的行业包括银行、高科技和生命科学。

然后是高盛的研究[来自高盛的研究](https://oreil.ly/3RHHw)，它预测生成式 AI 可能将全球 GDP 增加 7 万亿美元，并可能在下一个十年将生产力增长提高 1.5%。该公司的软件分析师 Kash Rangan 指出，“生成式 AI 可以简化业务工作流程，自动化常规任务，并催生新一代的商业应用程序。”

换句话说，企业将需要有效地采用和使用 AI 系统。这对他们可能是至关重要的。

# 考试详情

考试包含 65 个问题，其中 50 个计分，15 个不计分（这些不会公开）。不计分的问题不会影响你的最终分数，因为它们是实验性的。它们是 AWS 评估未来考试问题的方法。

无论怎样，猜测没有惩罚。如果你不确定答案，就尽你所能猜测。

考试包括不同类型的问题：

多选题

你将看到一个正确答案和三个错误选项，这些错误选项被称为*干扰项*。

多项选择

这些问题在五个或更多选项中有两个或更多正确答案。要获得分数，你需要每个都答对。

排序

你将获得三到五个选项来排列完成特定任务的正确顺序。

匹配题

你将匹配一组响应到三到七个提示。

案例研究

这些问题包含场景，包括两个或更多问题。每个问题单独评分。

考试结果以 100 到 1,000 分的分数表示。要及格，你需要至少 700 分。评分系统根据不同版本的考试进行调整，这些考试的难度略有不同。评分系统也是补偿性的。换句话说，你不需要在每个部分都获得及格分数。重要的是整体分数。

考试时长为 90 分钟，费用为 $100。然而，费用可能会根据你的位置和货币转换率而有所不同。

你可以选择在线或测试中心参加考试。无论哪种方式，都会有人监督考试，以确保没有作弊行为。

要注册考试，请访问 [*aws.training*](https://oreil.ly/zlHAM) 并创建账户。在顶部菜单中选择认证，然后选择考试注册。之后，点击“安排新考试”，选择你想要的考试，并选择“在测试中心现场”或“使用 OnVUE 在线”。然后你会看到安排页面。

如果你没有通过考试，你需要等待 14 天才能重新参加。没有次数限制，但每次都需要支付费用。

当你参加考试时，你不会立即看到你的通过/失败状态。相反，结果将在五个工作日内出现在你的 AWS 认证账户下的考试历史记录中。

该认证有效期为三年。在此之后，你需要重新参加考试。

# 其他认证

AWS 有四种类型的认证。首先，是基础级，不需要任何先前经验。AIF-C01 认证属于这一类别。其他还包括：

副级

这些是基于角色的，例如针对开发者、解决方案架构师或数据工程师。对于此类认证，AWS 建议你在该主题方面有经验。

专业级

这是为了那些想要获得应用或自动化流程设计高级技能的人。AWS 建议至少有两年的先前经验。

专项

这是你想要深入研究特定类别的时候。根据专业不同，有不同的经验推荐。
