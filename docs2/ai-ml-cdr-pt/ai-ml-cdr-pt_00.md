# 前言

亲爱的读者，

AI 正准备改变每一个行业，但几乎每个 AI 应用都需要根据其特定用途进行定制。用于阅读病历的系统与用于在工厂中寻找缺陷的系统不同，这与产品推荐引擎也不同。为了使 AI 充分发挥其潜力，工程师需要能够帮助他们适应数百万个具体问题中可用的惊人能力的工具。

当我领导谷歌大脑团队时，我们开始构建一个名为 DistBelief 的 C++深度学习框架。我们对于利用数千个 CPU 来训练神经网络（例如，使用 16,000 个 CPU 在未标记的 YouTube 视频上训练猫检测器）的潜力感到兴奋。自那时以来，深度学习已经走了多远！曾经的前沿技术现在可以在你的笔记本电脑上完成！在这本书中，你将学习如何使用 PyTorch 做到这一点。用于创建机器学习模型以实现 AI 的框架也已经走了很长的路。

PyTorch 取得了显著的进步。它旨在易于学习，同时足够强大，可供研究人员使用。其丰富的功能可以帮助人们构建各种 AI 工具，从简单的模型到从他人那里迁移学习，再到微调最现代的生成式 AI 模型。

今天，数百万的编码者因为像 PyTorch 这样的框架而成为了 AI 开发者。

拉罗伦斯·莫罗尼，一直是帮助开发者成功使用 TensorFlow 和 PyTorch 的主要力量。我有幸与他一起在 deeplearning.ai 和 Coursera 教授几个专业课程，包括即将推出的关于 PyTorch 的课程。

在我们合作的早期，拉罗伦斯曾给我发过一条 Slack 消息：

+   安德鲁唱了一首悲伤的老歌

+   晕倒在米勒小姐那里

+   邀请圈

+   晕倒，晕倒

+   [...]

他曾在传统爱尔兰歌曲的歌词上训练了一个 LSTM，并生成了这些诗句。但在 Transformer 网络时代之后，我们可以创建更加复杂的内容——从诗歌到代码，到对文档的分析！在这本书中，他涵盖了这两方面，并帮助你为现代 AI 做好准备。

如果 AI 能像那样开启乐趣的大门，谁会不愿意参与其中呢？你可以 1)从事推动人类进步的激动人心的项目，2)提升你的职业生涯，3)获得免费的爱尔兰诗歌。我祝愿你在学习 PyTorch 的旅程中一切顺利。

在拉罗伦斯的教导下，伟大的冒险等待着你们。
