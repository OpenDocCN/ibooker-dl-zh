# 前言

多年来，我沉浸在人工智能（AI）的世界中。我撰写了关于这个主题的几本书，并指导公司利用其变革潜力。在那段时间里，当我与人交谈时，一个问题成为了一个反复出现的主题：我如何提高我的 AI 技能？

我通常会分享一系列选项。有许多优秀的书籍、在线课程和 YouTube 教程可以帮助你入门。但如果你在寻找一个结构化的路径，我建议追求认证。认证不仅仅是测试你的知识——它们验证了你的专业知识。对于雇主来说，它们是一个信号，表明你不仅理解了基础知识，而且准备在现实世界中应用它们。

在众多 AI 认证中，有一个脱颖而出：Azure AI 基础（AI-900）考试。它不仅仅停留在表面。它深入到关键 AI 领域，如机器学习（ML）、深度学习（DL）、自然语言处理（NLP）、计算机视觉、生成式 AI 和负责任 AI。当然，还包括了来自 Microsoft Azure 的关键解决方案。

因此，无论你是规划新的职业道路还是加强当前的角色，AI-900 认证都能为你提供一个坚实的基础。这本书是你在这段旅程中的盟友。它包含了你需要通过考试的资源，也是更广泛 AI 主题的便捷参考指南。

因此，我的建议是：深入探索，保持好奇心，不要害怕挑战自己。AI 提供了无限的机会，有了正确的工具和心态，你将准备好产生影响。

感谢您选择这本书——祝您在 AI 冒险之旅中好运。

# 涵盖内容

下面是对每一章的简要概述。

第一章，“AI-900 考试简介”，概述了 Microsoft Azure AI-900 认证。本章突出了获得此认证的职业优势。它还详细介绍了考试结构以及关键主题，如 AI 工作负载和机器学习（ML）原则，并提供有关准备资源和互补认证的指导。

第二章，“Azure AI 服务”，介绍了 Azure 平台的基础知识以及如何设置它。本章还探讨了创建 Azure 资源，这是执行各种类型 AI 任务所必需的，我们还查看了一下 Azure AI Foundry，它允许你创建 AI 应用程序。

第三章，“AI 工作负载概述及关键用例”，探讨了各种 AI 工作负载及其实际应用。它介绍了基础技术，如内容审核、个性化、计算机视觉、NLP、知识挖掘、文档智能和生成式 AI。本章还强调了负责任的 AI 原则，包括公平性、可靠性、透明度和包容性。

第四章，“机器学习的基本原理”，考察了关键机器学习（ML）概念和技术。它描述了监督学习方法，如回归和分类，无监督技术，如聚类，以及机器学习（ML）与深度学习（DL）之间的区别。本章随后探讨了机器学习的工作流程，从数据准备和训练到推理。

第五章，“Azure 机器学习”，探讨了 Azure 的基于云的服务如何简化机器学习模型的训练、部署和管理。它突出了两个关键工具：用于自动化模型开发的 Azure Automated Machine Learning（AutoML）和用于创建管道的无代码、拖放界面的 Azure Machine Learning Designer。

第六章，“Azure 上计算机视觉工作负载的特点”，探讨了计算机视觉的基础知识。它涵盖了 Azure 的计算机视觉工具套件，包括用于通用图像分析的 Azure AI Vision、用于定制图像识别的 Azure AI Custom Vision 以及用于从图像中提取文本的光学字符识别（OCR）。本章详细介绍了图像分类、目标检测和面部分析等关键技术，并辅以实际案例。本章还解释了卷积神经网络（CNNs）在分析图像模式中的作用，并讨论了多模态模型。

第七章，“Azure 上自然语言处理工作负载的特点”，探讨了关键的自然语言处理（NLP）概念及其应用。它介绍了 Azure 的 NLP 服务，包括用于情感分析、实体识别和关键词提取的 Azure AI Language；用于实时文本翻译的 Azure AI Translator；以及用于语音到文本和文本到语音功能的 Azure AI Speech。本章还探讨了基础 NLP 技术，如分词、文本分类和语义语言模型。

第八章，“Azure 上生成式 AI 工作负载的特点”，突出了生成式 AI 的变革潜力。它侧重于 Azure 工具，如 OpenAI 模型、DALL-E 用于图像生成和 GPT-4 用于高级语言任务。关键概念包括大型语言模型（LLMs）、转换器架构、标记化、嵌入和注意力机制。

第九章，“成功通过 AI-900 考试的策略和技术”，提供了考试准备和测试的实用指导。它强调了利用微软考试沙盒来熟悉环境以及掌握关键主题（如 AI 基础、机器学习、自然语言处理、计算机视觉和生成式 AI）的重要性。
