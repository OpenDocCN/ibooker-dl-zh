# 第一部分：表格数据的机器学习介绍

本书的开篇部分旨在为你提供理解如何处理表格数据的基础。本部分章节探讨了表格数据的独特特征、不同的建模方法（机器学习与深度学习），以及探索性数据分析和准备的最佳实践。通过阅读这些章节，你将获得对处理表格数据特性的良好理解。你将为在后续部分处理更高级的技术做好准备。

第一章介绍了表格数据。本章解释了什么是表格数据，为什么它很重要，以及它与图像、音频和文本等其他类型数据的不同之处。我们还介绍了机器学习和深度学习概念，并试图解开关于在表格数据上使用深度学习方法的争议。本章最后通过回顾表格数据的独特特征，这些特征需要特定的分析方法来分析和建模。

第二章探讨了表格数据的结构和特征，突出了现实数据中常见问题的潜在问题和解决方案。我们提供了如何在在线和离线来源中找到表格数据的指导，特别是在商业组织中。本章还展示了如何进行有效的探索性数据分析的完整演示。

第三章基于三个关键标准——简单性、透明性和有效性——比较和对比了机器学习和深度学习方法，突出了每种方法的优点和缺点。我们还介绍了本书中将使用的 Airbnb 数据集。

完成第一部分后，你将准备好探索表格数据的经典和高级机器学习方法！
