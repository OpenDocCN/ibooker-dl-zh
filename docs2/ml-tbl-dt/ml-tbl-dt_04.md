# 第二部分：表格数据的机器学习和梯度提升

第四章、第五章和第六章全面概述了经典机器学习算法，并帮助您掌握最先进的梯度提升技术，如 XGBoost 和 LightGBM。您将学习如何使用每个算法并将其应用于合适的表格数据。第七章通过一个实际示例帮助您巩固所学知识，展示了涉及表格数据时的完整分析过程。

具体来说，第四章介绍了 Scikit-learn 以及各种经典机器学习方法，如线性回归、逻辑回归和广义线性模型。您将从实际角度掌握数据管道的工作原理，并学习如何验证结果以及在不同模型之间进行比较。然后我们进入第五章，探讨决策树及其集成，包括 bagging、随机森林和梯度提升决策树。接着，我们详细解释了梯度提升算法如何运作以及在表格数据上的优势。最后，第五章总结了从 Scikit-learn 到 XGBoost 和 LightGBM 的不同实现方法。

第六章致力于帮助您在机器学习中使用表格数据的最佳实践，特别强调梯度提升方法。我们涉及了许多高级技术，以选择最优的特征集、优化模型超参数以及获得更好的性能。我们还讨论了如何从实际角度处理缺失数据以及如何正确地转换分类数据。第二部分以第七章结束，这是一章说明性章节，您将学习如何使用梯度提升在真实世界数据上构建端到端示例。通过多个段落引导，您将掌握如何应用我们之前讨论的最佳实践和方法论。之后，您将准备好将这些经典和高级方法与深度学习方法进行比较，后者是以下结论部分的重点。
