# 第一部分 生成式 AI 的基础

本节介绍了支撑生成式 AI 的基本概念和技术。我们首先概述了生成式 AI 能做什么，它是如何工作的，以及如何在各种企业环境中应用。然后我们详细探讨了大型语言模型（LLMs）的细节，例如它们的结构、类别和主要概念。本节最后几章涵盖了通过 API 生成文本、图像和类似内容，提供了访问和利用这些技术的实用指南。

第一章介绍了生成式 AI 的概念，并解释了其创建新内容（如文本、图像和代码）的能力。它讨论了各种企业应用案例，比较了生成式 AI 与传统 AI，并为考虑采用这项技术的组织提供指导。

第二章深入探讨大型语言模型，解释其基础概念和架构，特别是关注 Transformer 模型。它还涵盖了不同类型的 LLM 以及诸如提示、令牌和嵌入等基本概念。

第三章探讨了如何使用 API 生成文本，从基本实现开始，逐步过渡到更高级的选项。它解释了模型类别和依赖关系，并提供了文本生成应用的实用示例。

第四章专注于生成图像和其他媒体类型。它概述了图像生成模型，并解释了如何使用 API 创建各种类型的媒体内容。

最后，第五章扩展到生成其他媒体类型，如视频和音频。它涵盖了媒体生成的技术和方法，提供了实际案例和 API 使用说明。本章还涵盖了使用各种模型进行代码生成。
