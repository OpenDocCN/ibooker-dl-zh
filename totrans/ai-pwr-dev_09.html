Part 4. Into the world

In part 4, we address the practical aspects of deploying and managing AI-integrated software in real-world environments. This section covers coding infrastructure and deployment strategies, from building Docker images to setting up continuous integration and continuous deployment (CI/CD) pipelines with tools like GitHub Actions. It also emphasizes secure application development, discussing threat modeling and the implementation of security best practices. Additionally, this part of the book explores the concept of democratizing access to AI by hosting your own LLM and utilizing platforms like GPT-4All. By providing practical guidance on deployment and security, this part prepares developers to successfully bring their AI-powered applications to market.
